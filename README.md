# Levelstead Site

Public website for [Levelstead](https://levelstead.com), including the product
overview, privacy policy, support page, and custom 404 page.

This repository intentionally contains no mobile-app source, customer data,
credentials, analytics, or backend services.

## Validate locally

```bash
node scripts/validate-site.mjs
```

GitHub Pages deploys `main` through the pinned workflow in
`.github/workflows/pages.yml`.
